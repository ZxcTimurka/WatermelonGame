1. Цель проекта:
Цель проекта - создать игру "WatermelonGame" на питоне с помощью PyGame.

2. Описание приложения:
Приложение состоит из следующих основных функциональных блоков:
1.Игра
2.Выбор дизайна
3.Таблица рейтинга

2.1. Типы пользователей:
Система предусматривает пользователя, который будет играть в игру.

2.2. Игра "WatermelonGame":
Игра чем-то напоминает тетрис. Цель игры с помощью соединения кругов одинакового цвета собрать самый большой.

2.4. Таблица результатов:
За соединения кругов будут даваться очки. Чем больше очков ты соберешь, тем выше ты будешь в таблице.

3. Предлагаемый стек технологий:
·Язык Python
·Библиотека PyGame
