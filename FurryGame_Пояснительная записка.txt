1. Цель проекта:
Создать игру на python с помощью библиотеки PyGame.
Описание приложения: 
1)Дизайн, понятный пользователю.
2)Смена скинов.
3)Таблица счета
2. Типы пользователей:
В приложении предусматривается пользователь.
2.2 Таблица вывода истории:
Приложение будет выводить базу данных с результатами прошлых игр.
3. Предлагаемый стек технологий:
1)Python
2)Библиотеки PyQt, PyGame, PyMunk, SQLite, random.
4.Требования к дизайну:
Удобный для пользователя
